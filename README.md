# Helse Sør-Øst Nano-learning Portal

## Formål
Dette prosjektet er en "Nano-learning Portal" designet for ansatte i Helse Sør-Øst (HSØ). Målet er å heve den digitale kompetansen gjennom korte, praktiske, og fokuserte læringsmoduler som kan gjennomføres i en travel sykehushverdag.

## Brukerhistorier
- **Som ansatt** ønsker jeg å kunne søke etter spesifikke problemer (f.eks. "hvordan låse PC") slik at jeg raskt kan løse hverdagsutfordringer.
- **Som ansatt** trenger jeg korte steg-for-steg instruksjoner fremfor lange kurs, slik at jeg kan lære mens jeg er på jobb.
- **Som innholdsprodusent** ønsker jeg at læringsinnholdet er skilt fra selve koden, slik at jeg enkelt kan oppdatere eller legge til nye moduler uten teknisk hjelp.

## Ikke-liste (I versjon 1)
- Ingen innlogging eller brukerprofiler (Applikasjonen er åpen).
- Ingen backend-database (data ligger i `mockData.json`).
- Ingen krevende video-streaming (vi bruker placeholders for video).
- Ingen sporing/analytikk av brukeradferd i første omgang.

## Teknisk Stack
- React / Vite
- Tailwind CSS
- React Router DOM
- Lucide React (Ikoner)

## Hvordan kjøre prosjektet
1. Åpne terminalen.
2. Kjør `npm install` for å installere avhengigheter.
3. Kjør `npm run dev` for å starte den lokale serveren.
